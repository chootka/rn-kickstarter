Connecting to your Raspberry Pi

There are a few ways you can connect to your Raspberry Pi - By plugging in an external monitor, keyboard and mouse, via SSH in the command line, or through an FTP client like Filezilla or Transmit. Read on...

Connect via SSH over the Raspberry Pi's WiFi Network
  • Join the 
    WHO'Z A GOOD BOI
    wireless network
  • Open a terminal and enter this command at the `$` prompt: 
    • $ ssh parrot@radical-network.local
    • password:
      toor
  • (Be aware that you will not be connected to the Internet!)
Connect via SFTP over the Raspberry Pi's WiFi Network
  • Join the 
    WHO'Z A GOOD BOI
    wireless network
  • Open up a FTP client like Transmit or Filezilla.
  • Make an SFTP connection with these details:
    • server: radical-network.local
    • port: 22
    • user: parrot
    • password: toor
Connect via external monitor + keyboard
  • Connect your Raspberry Pi via an HDMI cable to an HDMI monitor
  • Connect your Raspberry Pi to a keyboard and mouse via USB
  • Enter the password 
    toor
    for the 
    parrot
    user
Connect via Ethernet over SSH
  • Power on your RPi and connect an ethernet cable between your Raspberry Pi and computer. You may need an ethernet to USB or Thunderbolt adapter, depending on what kind of laptop you have.
  • Turn on internet sharing, so you will have access to the Internet while logged into your Raspberry Pi.
  • How to do it on OS X and Windows:
    • OS X
      • Open system preferences and select Sharing
      • On the right side of the screen, from the drop down select `Share your connection from: Wi-Fi`
      • Still on the right side of the screen, select `To computers using: Thunderbolt Ethernet` or it might be `Apple USB Ethernet Adapter`
      • On the left side of the panel, check the Internet Sharing box and click ‘Start’ in the prompt that comes up.
    • Windows 7
      • Go to your Control Panel
      • In the search box, type "Network Connections"
      • Find "View network connections" under Network and Sharing Center.
      • Enter it, and right click on the internet you would like to share and hit "Properties".
      • Go on the Sharing tab, and check "Allow other network users to connect through this Computer's internet connection"
      • Click Ok
      • Plug the ethernet cable into both of the computers and on the one you'd like to connect, it should automatically connect.
    • Windows 10
      • Click on the network icon in the lower right side of your screen and click on Open Network and Sharing Center
      • Click on Change adapter settings
      • Group select the Ethernet and Wi-Fi interfaces
      • Right click and choose ‘Bridge Connections'
  • Next, open a terminal and enter this command at the `$` prompt: 
    • $ ssh parrot@radical-network.local
    • password:
      toor

Basic Raspberry Pi Set-Up

Changing Your Password

After you've connected to your RPi, the first thing to do is to change the default password to something secure.

Via Command Line
  • $ sudo passwd parrot
  • $ <enter a new password>
  • $ <enter it again>
  • Done!
Via Desktop
  • Go to Applications > System Tools > MATE Terminal
  • $ sudo passwd parrot
  • $ <enter a new password>
  • $ <enter it again>
  • Done!

Changing Your Network Name (SSID)

After you've connected to your RPi, the first thing to do is to change the default password to something secure.

Via Command Line
  • $ sudo nano /etc/hostapd/hostapd.conf
  • Use the arrow key to move down to the `ssid` line. Change what comes after the `=` sign. Your wireless network can be up to 32 characters long.
  • $ sudo reboot
  • Done!
Via Desktop
  • Go to Applications > System Tools > MATE Terminal
  • $ sudo nano /etc/hostapd/hostapd.conf
  • Use the arrow key to move down to the `ssid` line. Change what comes after the `=` sign. Your wireless network can be up to 32 characters long.
  • $ sudo reboot
  • Done!

Adding a password to your wireless network

After you've connected to your RPi, the first thing to do is to change the default password to something secure.

Via Command Line
  • $ sudo nano /etc/hostapd/hostapd.conf
  • Use the arrow key to move down to the `wpa=0` line. Change this to `wpa=2`
  • Add the following lines underneath:
    • wpa_passphrase=<your password>
    • wpa_key_mgmt=WPA-PSK
  • Save and close the file (ctrl-x, then Y for yes to save)
  • $ sudo reboot
  • Done!
Via Desktop

Hosting Your Own Website

Replace this default site with your own
  • Open up a FTP client like Transmit or Filezilla.
  • Make an SFTP connection with these details:
    • Server: radical-network.local
    • User Name: parrot
    • Password: toor
    • Initial Path: /var/www/html
    • Port: 22
  • If you didn't set an `Initial Path`, navigate to /var/www/html/
  • You are now in your "web root", the location from which your website documents will be served. You can edit index.html, add new files, or delete everything and replace it with an entirely new site.
Change Your Hostname

Your hostname is currently radical-network. This is essentially the name of your device, which is automatically resolved to an IP address either in the browser window or in the command line. You can also think of it as the domain name associated with your offline webserver.

If you want to change this name to something else, it's easy. Follow these steps:

Via Command Line
  • Edit the hostname file by typing at the command prompt
    $ sudo nano /etc/hostname
  • Change `radical-network` to any other name you prefer.
  • Save and close the file (ctrl-x, then Y for yes to save)
  • Edit the hosts file by typing at the command prompt
    $ sudo nano /etc/hosts
  • Use the arrow keys to move down to the line that reads
    127.0.1.1   radical-network

    Change `radical-network` to whatever you specified in /etc/hostname.
  • Save and close the file (ctrl-x, then Y for yes to save)
  • $ sudo reboot
  • Done! You can now login to your RPi with
    $ ssh parrot@<your new hostname>.local

    or load your website at
    http://<your new hostname>.local
Via Desktop
  • Go to Applications > System Tools > MATE Terminal
  • Edit the hostname file by typing at the command prompt

Included Files & Tools

What's Inside

Your Raspberry Pi has Parrot OS installed on it. From wikipedia:

“Parrot Security OS (or ParrotSec) is a Linux distribution based on Debian with a focus on computer security. It is designed for penetration testing, vulnerability assessment and mitigation, computer forensics and anonymous web browsing.”

Parrot OS comes bundled with an array of network monitoring tools for you to explore. Anonymize yourself online, inspect WiFi traffic and more with:

  • Wireshark
  • nmap
  • macchanger
  • Ettercap
  • And many more...

From the Parrot OS desktop, find these application in Applications > Parrot > Information Gathering and Applications > Parrot > Sniffing & Spoofing
